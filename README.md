NOVA is a Python-based AI virtual assistant that performs voice-controlled tasks like opening applications and providing real-time information using speech recognition and text-to-speech technology.
 Features
- Voice command support
- Opens YouTube, WhatsApp, Instagram
- Tells current time
- Text-to-Speech responses
- Speech Recognition
Technologies Used
- Python
- SpeechRecognition
- pyttsx3
How to Run
1. Install required libraries:
   pip install pyttsx3 SpeechRecognition pyaudio
2. Run the assistant:
   python nova.py
